# Project: Coupler.io — Scheduler UX Review

## Scope

This project covers UX analysis of the Automatic Data Refresh (scheduler) feature
in Coupler.io. The primary analysis mode for this project is **Mode 3: UX / Feature review**.

## What we are reviewing

The scheduler configuration panel accessible from Data Flow → Settings.
This includes:
- The interval selector and its plan-based visibility
- The days-of-week selector and its interaction with sub-daily intervals
- The time preferences and timezone picker
- The toggle state and its visual representation of active / paused / failed states
- Failure and suspension behavior (how and where users are notified)
- Missing controls and unsupported business scenarios

## What is out of scope

- Backend pipeline execution correctness
- Source connector reliability
- Billing / payment flows
- Mobile app (web only)
