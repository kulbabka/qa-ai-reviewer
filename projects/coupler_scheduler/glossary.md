# Glossary — Coupler.io Scheduler

See product_context.md → Glossary section for the full term list.

Key terms to keep consistent in findings:

- Use **"data flow"**, not "importer" (the product's current preferred term)
- Use **"run"** for a single execution, not "sync" or "job"
- Use **"interval"** for the recurrence frequency setting
- Use **"toggle"** for the Automatic data refresh on/off switch
- Use **"plan limit"** when referring to scheduler restrictions tied to subscription tier
- Use **"row limit"** specifically for the per-run data volume cap (distinct from run quota)
